## commit 및 push 방법

- 본인의 이름(닉네임)으로 된 브랜치를 판다.
- 각 장의 폴더에 본인의 이름(닉네임)으로 폴더를 만들고, 정리한 내용을 .md파일로 올린다.
  - ex) 01_chapter > leedawn > 01_summary.md
- `git push`
- pr 날리기
  - 본인의 브랜치 -> main
